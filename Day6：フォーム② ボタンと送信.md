📘 Day6：フォーム② ボタンと送信

① 学習目的（なぜ学ぶ必要があるか）

フォーム入力に「実行ボタン」をつけることで、
ユーザーが意図的に操作できる UI を作れるようになるため。
行動追加画面や検索バーなど、アプリには必ず必要な機能。



② 例コード（意味を噛み砕いて説明）
<div>
  <h1>行動を追加</h1>
  <p>やる事を入力してください</p>
  <input type="text" placeholder="例：筋トレをする">

  <button>追加する</button>
</div>


解説（端的）

<input>：ユーザーが文字を入力する箱

<button>：ユーザーの「押す」という行動を受け取る

button があることで、UIが「操作可能なもの」になる



③ 手順（端的に）

<input> の下に <button> を追加

ボタンの中に操作の名前を書く（例：追加する）

<div> で全体をまとめて UI の形を整える

疑問：
余白はどう作るのですか？


余白は CSS の margin（外側の余白） を使って作ります。

あなたのコードに余白をつける最小の例を示します。

✅ 余白を作る最小コード例
<style>
  input {
    margin-bottom: 10px; /* 下に10pxの余白 */
  }
</style>

<div>
  <h1>行動を追加</h1>
  <p>やる事を入力してください</p>
  <input type="text" placeholder="例：筋トレをする">

  <button>追加する</button>
</div>

🔍 解説（端的）

margin-bottom: 10px;
→ input の 下に10pxの余白 を作る

<style>〜</style>
→ CSS を書く場所（Day8 で学ぶ）



💡 今理解しておけば十分

余白は margin

margin の値は px（ピクセル）

どの方向に余白をつけるかで名前が変わる

上：margin-top

下：margin-bottom

左：margin-left

右：margin-right
